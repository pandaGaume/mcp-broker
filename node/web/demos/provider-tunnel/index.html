Tunnel connection

This page hosts an MCP server built with the official @modelcontextprotocol/sdk. Connecting opens an outbound WebSocket to the broker's /provider/<name> endpoint — the broker then relays any MCP client that targets the slot above. The broker-specific code is isolated in js/lib/broker-tunnel.js.
