01 - Endpoints

Where to connect from

Four transports are mounted on the same host. Replace <name> with one of your provider slot names — or use _broker to talk to the broker itself.

Provider · WebSocket ws://localhost:3000/provider/<name>
Client · Streamable HTTP http://localhost:3000/<name>/mcp
Client · Raw WebSocket ws://localhost:3000/<name>
Client · Legacy SSE http://localhost:3000/<name>/sse

03 - Hide this page on next launch

Don't want a browser tab every time?

This landing only opens because www.open is true (the default for fresh installs). Flip it off and the broker will keep serving everything silently in the background.

Option A · config file

Set "open": false under www in .mcp-broker/config.json.

Option B · env var

Unset (or remove) MCP_BROKER_OPEN before launching the broker.

Option C · CLI flag

Pass --no-open on the command line — wins over both.

JSON · .mcp-broker/config.json 
{
  "port": 3000,
  "www": {
    "open": false,             // ← was true; flip to false
    "mounts": [{ "urlPrefix": "/", "dir": "www" }]
  }
}

Need the page back for a one-off? npx @cyanmycelium/mcp-broker --open overrides the config for a single launch.
